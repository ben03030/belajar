Penjelasan

Blender adalah aplikasi grafik komputer yang memungkinkan Anda untuk memproduksi suatu gambar atau animasi berkualitas tinggi dengan menggunakan geometri tiga dimensi.

Keunggulan

Kesederhanaan software ini membuat Blender tidak memerlukan spesifikasi komputer yang tinggi untuk sekedar menjalankannya. File instalasinya yang kecil membuat blender juga dapat diinstall dan dijalankan secara portable.

Operasi dasar pada software atau aplikasi Blender 3D cukup sederhana, yaitu hanya menggerakan objek, memutarkan objek, mengatur ukuran objek dan menambah objek. Selengkapnya simaklah penjelasan di bawah ini.

A. Movement


 Klik middle mouse lalu putar-2 object,
 Klik shift+middle mouse untuk geser2 object
 Scrolling mouse untuk zoom in - zoom out
 Tekan Ctrl dan tekan middle mouse terus maju mundur mouse untuk zoom in zoom out jadi lebih smooth

B. Position :


Yaitu operasi dasar untuk menggerakan objek baik itu untuk sumbu X, Y, dan Z maupun bebas. Menggunakan operasi ini dapat membuat benda berpindah tempat dari semulanya. Untuk melakukannya yaitu dengan cara menekan tombol G pada keyboard Anda.

Menggeser posisi pada sumbu X, Y atau Z :
G + X untuk menggeser pada posisi pada sumbu X
G + Y untuk menggeser pada posisi pada sumbu Y
G + Z untuk menggeser pada posisi pada sumbu Z

C. Rotate :


Yaitu untuk memutar objek supaya miring atau menghadap kebagian yang lain. Untuk melakukannya cukup menekan tombol R pada keyboard.

Memutar object di posisi pada sumbu X, Y atau Z :
R + X untuk memutar object pada posisi pada sumbu X
R + Y untuk memutar object pada posisi pada sumbu Y.
R + Z untuk memutar object pada posisi pada sumbu Z. Atau gunakan R 2x untuk memutar object lebih smooth

D. Scale :


Yaitu untuk mengatur atau mengubah ukuran objek atau benda untuk dapat diperbesar atau diperkecil dari ukuran semulanya. Untuk melakukannya tekan tombol S di keyboard.

Perbesar atau perkecil object pada sumbu X, Y atau Z :
S + X untuk memperbesar object disumbu X
S + Y untuk memperbesar object disumbu Y
S + Z untuk memperbesar object disumbu Z

E. Delete :


Untuk menghapus Objek cukup menekan tombol X pada keyboard.

F. NEW :


Untuk menambahkan objek baru tekan tombol SHIFT + A , akan tampil kotak dialog pop up, lalu pilih mesh dan pilih objek yang diinginkan mis : Plane, Cube, Circle, UV Sphere dll.
