Pengertian

Barisan adalah urutan bilangan yang memiliki karakteristik atau pola tertentu. Setiap bilangan yang ada pada barisan merupakan disebut suku dilambangkan Un dengan n adalah urutan suku.

Contoh

Barisan → U1 ,U2 ,U3 ,U4 , ... ,Un

Rumus suku ke-n barisan aritmatika

U1 = a U2 = U1 + b = a + b
U3 = U2 + b = (a + b) + b = a + 2b
U4 = U3 + b = (a + 2b) + b = a + 3b ... Un = U(n−1) + b = (a + (n − 2)b) + b = a + (n − 1)b

Un = a + (n − 1)b
Un = Suku ke− n
U1 = a = suku pertama
b = beda antar suku (b = Un − Un−1)
n = posisi suku

Pengertian

Barisan adalah urutan bilangan yang memiliki karakteristik atau pola tertentu. Setiap bilangan yang ada pada barisan merupakan disebut suku dilambangkan Un dengan n adalah urutan suku.

Contoh

Deret → Sn = U1 + U2 + U3 + U4 + ⋯ + Un = ∑ Ui

Rumus jumlah n suku pertama deret aritmatika

Sn = U1 + U2 + ...+ Un
Sn = a + (a + b) + (a + 2b) + ... + [a + (n – 1)b]
Sn = [a + (n – 1)b] + [a + (n – 2)b] + [a + (n – 3)b] + ... + a
----------------------------------------------------------------------------------- +
2Sn = [2a + (n – 1)b] + [2a + (n – 1)b] + [2a + (n – 1)b] + ... + [2a + (n – 1)b]
2Sn = n [2a + (n – 1)b]
Sn = n/2 [2a + (n – 1)b]

atau

Sn = n/2 [a + a + (n – 1)b)]
Sn = n/2 (a + Un)

Sn = n/2 [2a + (n – 1)b] atau Sn = n/2 (a + Un)
Sn = jumlah n suku pertama
U1 = a = S = suku pertama
b = beda antar suku
n = posisi suku
